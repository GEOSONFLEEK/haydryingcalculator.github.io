Погода вручную

Задай ожидаемую погоду сам — удобно взять с Gismeteo или из «Погоды»: там на карточке дня есть «днём / ночью». Приложение построит суточный ход температуры. Это усреднение по одному профилю на все дни; для разной погоды по дням лучше онлайн-прогноз или встроенный снимок.
Параметры расчёта ▾
Модель: W(t) = W₀ − Σ (KT·T + KV·V)/24 по каждому часу прогноза (T — °C, V — м/с). Коэффициенты KT, KV — ориентировочные значения по умолчанию; их стоит откалибровать под свои полевые замеры или агрономические таблицы для конкретной культуры и региона.
Данные о погоде: онлайн через Open-Meteo (когда открыто по ссылке https), иначе — встроенный снимок прогноза по Калининграду (10 июл).
На телефоне из локального файла интернет недоступен — работает встроенный снимок; для живого прогноза открой приложение по адресу https или обнови снимок.
